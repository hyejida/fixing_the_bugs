Boardgame CSS Animations

Project Details

Scope: Web Design, Web Development

Tools Used:HTML, CSS, JavaScript, Adobe Illustrator

Timeline: 2 Week

Using CSS, we can create beautiful animations of out HTML elements without using JavaScript. We can use the built-in @keyframe at-rule, which controls the steps of an animated sequence by defining the style of each keyframes.

Link to a Live Server

CSS Animations

Make a gird that is a set of intersecting six columns and four rows. Grid was designed for two-dimensional layout - rows, and columns at the same time.

Set a grid position with the grid-row and grid-column properties. Then, apply a transformation that rotates an element.

Use the keyframes and transform CSS properties to create an anitmated Intro. With preserve-3dsyntax, indicates the children of the element should be positioned in the 3D-space

I made every grid tiles with the <`input type="radio"> to be selected. Then used ::before selector with a cute rabbit character to insert cotent before a selected radio button.
